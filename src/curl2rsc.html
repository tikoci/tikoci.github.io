    curl2rsc browser-side /tool/fetch generator

    Paste a curl command and convert it into RouterOS /tool/fetch output without leaving the browser.

    How to use this tool? …
    What works today

    Paste or edit a curl command. The page parses it live, shows warnings, and generates RouterOS /tool/fetch output entirely in your browser.

    What to watch for

    Modern curl options like --json are handled directly. Redirect-following, compressed responses, and local file uploads are called out when RouterOS cannot mirror curl exactly.

    Sharing safely

    The share button copies the current URL, including the curl text in query params. Redact passwords, tokens, or cookies before sharing.

    Behind the curtain — parsing and RouterOS generation both happen fully in the browser using the bundled curlconverter wrapper, Tree-sitter WASM assets, and a local RouterOS generator extracted from the old Postman-based implementation. No conversion backend is contacted.
